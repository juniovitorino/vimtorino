# Neovim Configuration Files

This repository is my Neovim Editor configuration files. Here you find all needed configurations to work with the Ruby, PHP, Typescript, Javascript, Elixir languages and with frameworks/libraries React, React Native, Vue, Svelte, Next.js, Nuxt.js, Ruby on Rails, Phoenix and Node.js.



## Requirements

 - Nvim 5.0+ - https://github.com/neovim/neovim
 - LazyGit - https://github.com/jesseduffield/lazygit
 - Ripgrep (RG) - https://github.com/BurntSushi/ripgrep


## Installation

In order to install vimtorino you should clone this repo inside your `$HOME/.config` files with **nvim** folder name.

```bash
mv ~/.config/nvim ~/.config/nvim.bak && git clone https://github.com/juniovitorino/vimtorino.git ~/.config/nvim
```
